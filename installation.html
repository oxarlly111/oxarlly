Oxarlly Installation Guide

Follow the steps below to install and get started with Oxarlly.

Step 1: Install Oxarlly Package

To install Oxarlly, run the following command in your terminal:

npm install oxarlly-temp

Step 2: Verify Installation

Once the package is installed, you can verify that it was installed correctly by checking your node_modules folder or by running the following command:

npm list oxarlly-temp

That's it! Now you can move on to using Oxarlly.
