Oxarlly Usage Guide

Follow the instructions below to use Oxarlly and generate your range of numbers.

Step 1: Create the Required Text Files

First, create the following text files:

  • start.txt - Contains the start value.
  • end.txt - Contains the end value.
  • step.txt (optional) - Contains the step value. Defaults to 1 if not provided.

Step 2: Run the Generator

Once the files are in place, run the following command in your terminal:

node src/index.js

Step 3: View the Results

The generated numbers will be saved to result.txt. You can open this file to view the range of numbers.

That's it! You have successfully used Oxarlly to generate a range of numbers based on your input.
